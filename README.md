# D&R VUE

Vue.js, nuxt.js teknolojileri kullanılarak D&R e-ticaret sitesinin web sitesinin geliştirilmesi.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
